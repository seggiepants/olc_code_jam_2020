The Great Machine Armada

by: seggiepants
Created for the OLC Code Jam 2020.
Controls:
  • A or left arrow key to move the tank left
  • D or right arrow key to move the tank right
  • Space Bar to fire the laser
  • P to Pause/Unpause the game
    • Escape to exit the game (no prompt).
